This page presents the key artifacts from the WeightWatcher Android application, showing both original and enhanced code. Each artifact demonstrates skills in software engineering, data structures, algorithms, database optimization, and security principles.

MainActivity.java (Software Engineering and Design)

Description: The MainActivity.java file handles user authentication and navigation. Originally, it had duplicated code and lacked robust exception handling. The enhanced version refactors the code, improves error handling, and adds clear documentation.

DataDisplayActivity.java (Algorithms and Data Structures)

Description: This activity displays and manages weight entries, goal weights, and user progress. Enhancements include improved exception handling, better data retrieval logic, and performance optimizations for displaying lists.

  • Original DataDisplayActivity
  • Enhanced DataDisplayActivity

DataAdapter.java (User Experience and Performance)

Description: The DataAdapter manages weight entries in a RecyclerView. The enhanced version adds exception handling, removes unnecessary notify calls to improve performance, and introduces a confirmation dialog to enhance user experience.

  • Original DataAdapter
  • Enhanced DataAdapter

AddEntryDialog.java (Input Validation and Error Handling)

Description: This dialog fragment collects user weight input. Originally, it lacked proper input validation and error handling. The enhanced version validates user input, handles NumberFormatException, and provides informative feedback.
